This repo contains uploads of macros I've written for foundryvtt that might be useful for others.

Currently, it contains:
* __loot_roll.js__: A script designed to perform loot rolls for a battle-royale type dnd contest. Heavily configurable, with various item types and modifiers, designed to base the roll quality off of player's investigation checks. To use this, you need to create and configure trinket tables for it to roll on.
